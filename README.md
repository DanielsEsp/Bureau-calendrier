# Bureau-calendrier

Nouveau projet avec mon écran favori.  Un module de bureau affichant un calendrier mensuel avec horloge et mini station météo sur base Météo API et l'affichage des 4 événements de mon agenda Google pour le jour J et J+1. Toutes ces info sont parfaitement lisible sur cet écran e-Paper 4.26' 800x400 pixels.
